## Game Class
- 입력받은 사용자의 이름을 게임할 수 있는 형태로(Ex. List<Car>) 변환하는 기능
- 게임 시작 기능
- 최종 우승자 출력 기능 
- 매 횟수 별로 이동 거리 출력 기능

## Car Class
- 자동차 전진 기능
    - 랜덤 번호(0-9) 뽑기 기능

## Validate
- 문자나 쉼표가 아닌 다른 문자가 들어갔을 때 예외 처리 기능
- 사용자의 이름이 5글자 초과일 때 예외 처리 기능
- 사용자의 이름이 공백일 떄 예외 처리 기능
- 시도할 횟수에 숫자가 아닌 다른 값이 입력되었을 때 예외 처리 기능

## Util
- 입력 기능

## Test
